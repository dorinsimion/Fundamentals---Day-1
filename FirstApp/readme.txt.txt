Tipurile de date primitive:

Pentru stocare variabile de tip intreg:
	byte
	short
	int
	long
Pentru stocarea numerelor reale:
	float
	double

Pentru stocarea de valori true sau false
	boolean

Pentru stocarea de caractere unicode (un singur caracter)
	char

Numele de variabile:
	pot incepe cu o litera, _ sau $
	nu pot contine caractere special (&,\.@)
	nu pot fii keyworduri (cuvinte rezervate de limbaj; exemplu : void, public,private)
