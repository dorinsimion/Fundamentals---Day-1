Structuri decizionale:
-if
-switch

Structura instructiunii if:

if(conditie1){
	//bloc instructiuni
} else if(conditie2){
	//bloc instructiuni	
} else {
	//bloc instructiuni
}

Pot exista 0 sau mai multe blocuri else if
Poate exista una sau nici o instructiune else
Acoladele pot lipsi daca avem o singura instructiune(daca sunt mai multe instructiuni, sunt obligatorii). Se recomanda totusi sa le folosim si in acest caz.
